### 🔍 AllKeyShop Price Monitor

**Version:** 1.2.0
**Autor:** Sassoanarchico
**Typ:** Generic Plugin
**Status:** ✅ Working (Done)
**Sprachen:** 🇬🇧 EN · 🇮🇹 IT · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE

Überwache Spielpreise auf AllKeyShop und erhalte Benachrichtigungen über kostenlose Spiele.

#### ✨ Hauptfunktionen

- **Preisüberwachung**: Füge Spiele zur Überwachung hinzu und sieh den niedrigsten Preis auf AllKeyShop
- **Automatische Aktualisierung**: Preise werden automatisch in konfigurierbaren Intervallen aktualisiert
- **Kostenlose-Spiele-Benachrichtigungen**: Erhalte Benachrichtigungen, wenn neue kostenlose Spiele gefunden werden
- **Mehrere Plattformen**: Unterstützt Steam, Epic, GOG, Xbox, PlayStation und mehr
- **Seitenleiste**: Dediziertes Seitenleisten-Panel mit überwachten Spielen, Preisen und Gratisspielen auf einen Blick
- **Doppelte Preisschwellen**: Separate Schwellen für Key- und Account-Preise mit unabhängigen Benachrichtigungen
- **Spielsymbole**: AllKeyShop-Thumbnails in der Liste der überwachten Spiele
- **Vollständige Lokalisierung**: Oberfläche in 5 Sprachen übersetzt (~120 Lokalisierungsschlüssel)

### ⚙️ Automation Profile Manager

**Version:** 0.5.0
**Autor:** sassoanarchico
**Typ:** Generic Plugin
**Status:** 🚧 Work in Progress
**Sprachen:** 🇬🇧 EN · 🇮🇹 IT · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE

Verwalte Automatisierungsprofile für deine Spiele mit wiederverwendbaren Aktionen. Führe automatisch Aktionen beim Starten oder Schließen eines Spiels aus.

#### ✨ Hauptfunktionen

- **~50 Standardaktionen**: Erweiterte Bibliothek mit Aktionen für Browser, Kommunikation, Gaming, System und mehr
- **Focus Assist**: Aktiviert/deaktiviert den Windows Focus Assist Modus automatisch
- **App-Verwaltung**: Schließt/startet gängige Apps automatisch (Chrome, Brave, Discord, Spotify, Zoom, WhatsApp, Signal, Notion)
- **Auflösungswechsel**: Unterstützung für 4K, 1080p, 720p mit automatischer Wiederherstellung
- **Systembefehle**: Energiepläne, Wartezeiten, PowerShell-Skripte, Game Mode
- **Mirror-Aktionen**: Stellt automatisch den vorherigen Zustand beim Beenden des Spiels wieder her
- **Wiederverwendbare Profile**: Erstelle Profile und weise sie einem oder mehreren Spielen zu
- **Emulator-Vorlagen**: Fertige Profile für RetroArch, Dolphin, PCSX2, PPSSPP, Cemu
- **Setup-Assistent**: Geführte Konfiguration in 6 Schritten mit automatischer Erkennung installierter Apps
- **Seitenleiste**: Durchsuche alle Profile und ihre zugewiesenen Spiele direkt in der Seitenleiste; weise Spiele zu oder entferne sie, ohne die Einstellungen zu öffnen
- **Kontextmenüs**: Weise Profile direkt über das Rechtsklick-Menü des Spiels zu/entferne/zeige sie an
- **Vollständige Lokalisierung**: Oberfläche in 5 Sprachen übersetzt (~265 Lokalisierungsschlüssel)

### 🚀 FastInstall

**Version:** 1.3.5
**Autor:** sassoanarchico
**Typ:** Game Library
**Status:** ⚠️ Working (Bugfixing)

Archiviere Spiele auf einer langsamen HDD und installiere sie schnell auf einer SSD, indem du die Dateien kopierst, wenn du spielen möchtest.

#### ✨ Hauptfunktionen

- **Intelligente Archivierung**: Archiviere Spiele auf HDD und installiere sie bei Bedarf auf SSD
- **Hintergrundinstallation**: Nicht blockierender Prozess mit Fortschrittsfenster
- **Download-Warteschlange**: Unterstützung für mehrere Downloads mit Warteschlange und Priorität
- **Parallele Downloads**: Konfiguriere 1-10 gleichzeitige Installationen
- **Archiv-Unterstützung**: Automatische Extraktion von ZIP, RAR und 7Z
- **Integritätsprüfung**: Überprüft die Dateiintegrität nach dem Kopieren
- **Speicherplatzprüfung**: Überprüft den verfügbaren Speicherplatz vor der Installation
- **Emulator-Unterstützung**: Emulator-Konfiguration für Konsolenplattformen
- **Lokalisierung**: Vollständige Unterstützung für Englisch und Italienisch

### 🎯 Myrient Library

**Version:** 0.15.0
**Autor:** sassoanarchico
**Typ:** Game Library
**Status:** 🚧 Work in Progress

Playnite-Erweiterung zum Suchen, Importieren und Herunterladen von ROMs aus dem Myrient-Archiv.

#### ✨ Hauptfunktionen

- **ROM-Suche**: Suche einzelne ROMs auf Myrient mit Plattformfilter
- **Massenimport**: Importiere alle ROMs einer bestimmten Sammlung/Plattform
- **Direkter Download**: Lade ROMs direkt von Myrient herunter, wenn du auf „Installieren" klickst
- **Dateiorganisation**: Organisiere heruntergeladene Dateien nach Plattform und/oder Sammlung
- **Integritätsprüfung**: Überprüfe die Integrität heruntergeladener Dateien per Hash
- **Download-Warteschlange**: Unterstützung für mehrere Downloads mit Warteschlange
- **Unterstützte Sammlungen**: Unterstützung für No-Intro, Redump und andere Sammlungen
- **Seitenleiste**: Dediziertes Seitenleisten-Panel mit aktiven Downloads und deiner Myrient-Bibliothek; suche, installiere und deinstalliere Spiele direkt im Panel

### 📂 EmuLibrary (Fork)

**Version:** 3.5.0
**Autor:** psychonic (Fork von sassoanarchico)
**Typ:** Game Library
**Status:** ✅ Working

Fork von EmuLibrary mit Unterstützung für den neuen ROM-Typ **RyujinxCopy**, entwickelt zur Verwaltung von Nintendo Switch-Spielen über den Ryujinx-Emulator.

#### ✨ Hauptfunktionen

- **RyujinxCopy ROM-Typ**: Kopiert vollständige Spielordner (Basis + Updates + DLC) vom Quell- zum Ziellaufwerk
- **Automatische Update/DLC-Registrierung**: Konfiguriert Ryujinx automatisch zum Laden von Updates und DLCs bei der Installation
- **Automatische Deregistrierung**: Entfernt Konfigurationen aus Ryujinx, wenn das Spiel deinstalliert wird
- **Intelligentes Path-Rewriting**: Handhabt Laufwerks-/Pfadänderungen durch Umschreiben bestehender Konfigurationen
- **Basisspiel-Erkennung**: Identifiziert automatisch die Basisdatei über `[v0]`, Title ID endend auf `000`, oder größte Datei
- **Namensbereinigung**: Extrahiert saubere Namen aus getaggten Ordnern wie `"Game (NSP)(EU)(Base Game)"`
- **Windows-Kopierdialog**: Verwendet den nativen Windows-Kopierdialog während der Installation

## 🛠️ Voraussetzungen

Alle Erweiterungen benötigen:
- **Playnite** Version 9.x oder höher (einige erfordern 10.x)
- **.NET Framework** 4.8
- **Internetverbindung** (für die meisten Funktionen)

## 📝 Installationshinweise

1. Lade die `.pext`-Datei von der Release-Seite der gewünschten Erweiterung herunter
2. Öffne Playnite
3. Gehe zu `Menü > Erweiterungen > Erweiterung installieren`
4. Wähle die heruntergeladene `.pext`-Datei aus
5. Starte Playnite neu, wenn aufgefordert

### ⚙️ Automation Profile Manager

**Version:** 0.6.0
**Author:** sassoanarchico
**Type:** Generic Plugin
**Status:** ✅ Working (Done)
**Languages:** 🇬🇧 EN · 🇮🇹 IT · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE

Manage automation profiles for your games with reusable actions. Automatically execute actions when you start or close a game.

#### ✨ Main Features

- **~50 Default Actions**: Expanded library with actions for browsers, communication, gaming, system and more
- **Focus Assist**: Toggle Windows Focus Assist mode automatically
- **App Management**: Automatically close/start common apps (Chrome, Brave, Discord, Spotify, Zoom, WhatsApp, Signal, Notion)
- **Resolution Switching**: 4K, 1080p, 720p support with automatic restore
- **System Commands**: Power plans, waits, PowerShell scripts, Game Mode
- **Mirror Actions**: Automatically restore state when the game closes
- **Reusable Profiles**: Create profiles and assign them to one or more games
- **Emulator Templates**: Ready-made profiles for RetroArch, Dolphin, PCSX2, PPSSPP, Cemu
- **Wizard Setup**: Guided setup in 6 steps with automatic detection of installed apps
- **Sidebar Panel**: Browse all profiles and their assigned games directly from the sidebar; assign or remove games without opening settings
- **Context Menus**: Assign/remove/view profiles directly from game right-click menus
- **Full Localization**: Interface translated into 5 languages (~265 localization keys)

### 🚀 FastInstall

**Version:** 1.3.5
**Author:** sassoanarchico
**Type:** Game Library
**Status:** ✅ Working (Done)

Archive games on a slow HDD and install them quickly on an SSD by copying files when you want to play.

#### ✨ Main Features

- **Smart Archiving**: Archive games on HDD and install on SSD when needed
- **Background Installation**: Non-blocking process with progress window
- **Download Queue**: Support for multiple downloads with queue and priority

- **Archive Support**: Automatic extraction of ZIP, RAR and 7Z
- **Integrity Check**: Verify file integrity after copy
- **Disk Space Check**: Verify available space before installation
- **Emulator Support**: Emulator configuration for console platforms

### 🎯 Myrient Library

**Version:** 0.17.0
**Author:** sassoanarchico
**Type:** Game Library
**Status:** ✅ Working (Done)

Playnite extension to search, import and download ROMs from the Myrient archive.

#### ✨ Main Features

- **ROM Search**: Search single ROMs on Myrient with platform, language and type filtering
- **Search Filters**: Filter search results by language, hide demos, and hide beta/alpha/prototype ROMs
- **Bulk Import**: Import all ROMs from a specific collection/platform
- **Direct Download**: Download ROMs directly from Myrient via aria2c with 16 parallel connections
- **Download Progress**: Real-time progress window with speed, average speed and ETA
- **Download Completion**: Progress window shows completion summary with final stats before closing
- **Theme Support**: Download UI adapts automatically to your Playnite theme (dark/light)
- **File Organization**: Organize downloaded files by platform and/or collection
- **Auto-Extraction**: ZIP archives extracted automatically after download
- **Nested ROM Detection**: Handles archives with nested folders (e.g. game.zip > game/ > game.nds)
- **Emulator Config**: Per-platform emulator setup applied automatically after install
- **Supported Collections**: Support for No-Intro, Redump and other collections
- **Platform Cleanup**: Invalid platform entries (e.g. /README, /files) filtered automatically
- **Sidebar Panel**: Dedicated sidebar showing active downloads and your Myrient game library

### 📂 EmuLibrary (Fork)

**Version:** 3.5.0
**Author:** psychonic (Fork by sassoanarchico)
**Type:** Game Library
**Status:** ✅ Working

Fork of EmuLibrary with support for the new **RyujinxCopy** ROM type, designed to manage Nintendo Switch games through the Ryujinx emulator.

#### ✨ Main Features

- **RyujinxCopy ROM Type**: Copy entire game folders (base + updates + DLC) from source to destination disk
- **Automatic Update/DLC Registration**: Automatically configures Ryujinx to load updates and DLCs at install time
- **Automatic Deregistration**: Removes configurations from Ryujinx when the game is uninstalled
- **Smart Path Rewriting**: Handles drive/path changes by rewriting existing configs
- **Base Game Detection**: Automatically identifies the base file via `[v0]`, Title ID ending in `000`, or largest file
- **Name Cleanup**: Extracts clean names from tagged folders like `"Game (NSP)(EU)(Base Game)"`
- **Windows Copy Dialog**: Uses the native Windows copy dialog during installation

## 📝 Installation Notes

1. Download the `.pext` file from the release page of the desired extension
2. Open Playnite
3. Go to `Menu > Extensions > Install Extension`
4. Select the downloaded `.pext` file
5. Restart Playnite when prompted

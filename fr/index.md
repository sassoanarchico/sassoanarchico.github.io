### 🔍 AllKeyShop Price Monitor

**Version :** 1.2.0
**Auteur :** Sassoanarchico
**Type :** Generic Plugin
**Statut :** ✅ Working (Done)
**Langues :** 🇬🇧 EN · 🇮🇹 IT · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE

Surveillez les prix des jeux sur AllKeyShop et recevez des notifications sur les jeux gratuits.

#### ✨ Fonctionnalités Principales

- **Surveillance des Prix** : Ajoutez des jeux à surveiller et consultez le prix le plus bas trouvé sur AllKeyShop
- **Mise à Jour Automatique** : Les prix se mettent à jour automatiquement à des intervalles configurables
- **Alertes Jeux Gratuits** : Recevez des notifications lorsque de nouveaux jeux gratuits sont trouvés
- **Plateformes Multiples** : Compatible avec Steam, Epic, GOG, Xbox, PlayStation et plus
- **Panneau Latéral** : Panneau latéral dédié affichant les jeux surveillés, prix et alertes de jeux gratuits en un coup d'œil
- **Double Seuil de Prix** : Seuils séparés pour les prix Key et Account avec notifications indépendantes
- **Icônes de Jeux** : Miniatures AllKeyShop dans la liste des jeux surveillés
- **Localisation Complète** : Interface traduite en 5 langues (~120 clés de localisation)

### ⚙️ Automation Profile Manager

**Version :** 0.6.0
**Auteur :** sassoanarchico
**Type :** Generic Plugin
**Statut :** ✅ Working (Done)
**Langues :** 🇬🇧 EN · 🇮🇹 IT · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE

Gérez des profils d'automatisation pour vos jeux avec des actions réutilisables. Exécutez automatiquement des actions au lancement ou à la fermeture d'un jeu.

#### ✨ Fonctionnalités Principales

- **~50 Actions par Défaut** : Bibliothèque étendue avec actions pour navigateurs, communication, gaming, système et plus
- **Focus Assist** : Active/désactive le mode Focus Assist de Windows automatiquement
- **Gestion d'Apps** : Ferme/lance automatiquement les apps courantes (Chrome, Brave, Discord, Spotify, Zoom, WhatsApp, Signal, Notion)
- **Changement de Résolution** : Support 4K, 1080p, 720p avec restauration automatique
- **Commandes Système** : Plans d'alimentation, attentes, scripts PowerShell, Game Mode
- **Actions Mirror** : Restaure automatiquement l'état à la fermeture du jeu
- **Profils Réutilisables** : Créez des profils et assignez-les à un ou plusieurs jeux
- **Modèles Émulateurs** : Profils prêts à l'emploi pour RetroArch, Dolphin, PCSX2, PPSSPP, Cemu
- **Assistant de Configuration** : Configuration guidée en 6 étapes avec détection automatique des apps installées
- **Panneau Latéral** : Parcourez tous les profils et leurs jeux assignés directement depuis la barre latérale ; assignez ou supprimez des jeux sans ouvrir les paramètres
- **Menus Contextuels** : Assignez/supprimez/consultez les profils directement depuis le menu clic droit du jeu
- **Localisation Complète** : Interface traduite en 5 langues (~265 clés de localisation)

### 🚀 FastInstall

**Version :** 1.3.5
**Auteur :** sassoanarchico
**Type :** Game Library
**Statut :** ✅ Working (Done)

Archivez vos jeux sur un HDD lent et installez-les rapidement sur un SSD en copiant les fichiers quand vous voulez jouer.

#### ✨ Fonctionnalités Principales

- **Archivage Intelligent** : Archivez les jeux sur HDD et installez-les sur SSD au besoin
- **Installation en Arrière-plan** : Processus non bloquant avec fenêtre de progression
- **File de Téléchargement** : Support pour téléchargements multiples avec file et priorité
- **Téléchargements Parallèles** : Configuration de 1 à 10 installations simultanées
- **Support d'Archives** : Extraction automatique de ZIP, RAR et 7Z
- **Vérification d'Intégrité** : Vérifie l'intégrité des fichiers après la copie
- **Contrôle d'Espace Disque** : Vérifie l'espace disponible avant l'installation
- **Support Émulateurs** : Configuration d'émulateurs pour les plateformes console
- **Localisation** : Support complet pour l'anglais et l'italien

### 🎯 Myrient Library

**Version :** 0.17.0
**Auteur :** sassoanarchico
**Type :** Game Library
**Statut :** ✅ Working (Done)

Extension Playnite pour rechercher, importer et télécharger des ROMs depuis l'archive Myrient.

#### ✨ Fonctionnalités Principales

- **Recherche de ROMs** : Recherchez des ROMs individuelles sur Myrient avec filtre par plateforme, langue et type
- **Filtres de Recherche** : Filtrez les résultats par langue, masquez les démos et masquez les ROMs bêta/alpha/prototype
- **Importation en Masse** : Importez toutes les ROMs d'une collection/plateforme spécifique
- **Téléchargement Direct** : Téléchargez des ROMs directement depuis Myrient via aria2c avec 16 connexions parallèles
- **Progression du Téléchargement** : Fenêtre en temps réel avec vitesse, vitesse moyenne et temps restant
- **Fin de Téléchargement** : La fenêtre de progression affiche un résumé final avec les statistiques avant la fermeture
- **Support des Thèmes** : L'interface de téléchargement s'adapte automatiquement au thème Playnite (clair/sombre)
- **Organisation des Fichiers** : Organisez les fichiers téléchargés par plateforme et/ou collection
- **Extraction Automatique** : Les archives ZIP sont extraites automatiquement après le téléchargement
- **Détection ROMs Imbriquées** : Gère les archives avec dossiers imbriqués (ex. game.zip > game/ > game.nds)
- **Config Émulateur** : Configuration émulateur par plateforme appliquée automatiquement après l'installation
- **Collections Supportées** : Support pour No-Intro, Redump et autres collections
- **Nettoyage des Plateformes** : Les entrées de plateforme invalides (ex. /README, /files) sont filtrées automatiquement
- **Panneau Latéral** : Panneau latéral dédié avec téléchargements actifs et votre bibliothèque Myrient

### 📂 EmuLibrary (Fork)

**Version :** 3.5.0
**Auteur :** psychonic (Fork par sassoanarchico)
**Type :** Game Library
**Statut :** ✅ Working

Fork d'EmuLibrary avec support du nouveau type de ROM **RyujinxCopy**, conçu pour gérer les jeux Nintendo Switch via l'émulateur Ryujinx.

#### ✨ Fonctionnalités Principales

- **Type ROM RyujinxCopy** : Copie des dossiers de jeu complets (base + mises à jour + DLC) du disque source au disque de destination
- **Enregistrement Automatique Update/DLC** : Configure automatiquement Ryujinx pour charger mises à jour et DLCs à l'installation
- **Désenregistrement Automatique** : Supprime les configurations de Ryujinx quand le jeu est désinstallé
- **Réécriture Intelligente des Chemins** : Gère le changement de lecteurs/chemins en réécrivant les configs existants
- **Détection du Jeu de Base** : Identifie automatiquement le fichier de base via `[v0]`, Title ID se terminant par `000`, ou fichier le plus volumineux
- **Nettoyage des Noms** : Extrait des noms propres depuis des dossiers tagués comme `"Game (NSP)(EU)(Base Game)"`
- **Dialogue de Copie Windows** : Utilise le dialogue de copie natif de Windows pendant l'installation

Toutes les extensions nécessitent :
- **Playnite** version 9.x ou supérieure (certaines nécessitent 10.x)
- **.NET Framework** 4.8

## 📝 Notes d'Installation

1. Téléchargez le fichier `.pext` depuis la page de releases de l'extension souhaitée
2. Ouvrez Playnite
3. Allez dans `Menu > Extensions > Installer une extension`
4. Sélectionnez le fichier `.pext` téléchargé
5. Redémarrez Playnite quand demandé
